# Data

Historical daily weather observations for London from the European Climate Assessment & Dataset (ECA&D), covering 1979–2020.

## Columns

| Column | Description |
|--------|-------------|
| date | Date (YYYYMMDD) |
| cloud_cover | Cloud cover (oktas) |
| sunshine | Sunshine duration (hours) |
| global_radiation | Global radiation (W/m²) |
| max_temp | Maximum temperature (°C) |
| mean_temp | Daily mean temperature (°C) — prediction target |
| min_temp | Minimum temperature (°C) |
| precipitation | Precipitation (mm) |
| pressure | Sea-level pressure (hPa) |
| snow_depth | Snow depth (cm) |
